# 入门测试题目
## 题目背景

强烈推荐[新用户必读贴](/discuss/show/241461)
## 题目描述

求两个整数的和。
## 输入格式

一行，两个用空格隔开的整数。
## 输出格式

两个整数的和。
## 样例

### 样例输入 #1
```
1 2
```
### 样例输出 #1
```
3
```
### 样例输入 #2
```
10230 21312
```
### 样例输出 #2
```
31542
```
## 提示

对于 $100\%$ 的数据，输入的整数在 $[1, 4 \times {10}^{18}]$ 内。
