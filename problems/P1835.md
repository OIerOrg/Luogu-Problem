# 素数密度
## 题目背景

UPD:

- 2024.8.12：加入一组 Hack 数据。
## 题目描述

 给定 $L,R$，请计算区间 $[L,R]$ 中素数的个数。

   $1\leq L\leq R < 2^{31}$，$R-L\leq 10^6$。
## 输入格式

第一行，两个正整数 $L$ 和 $R$。
## 输出格式

一行，一个整数，表示区间中素数的个数。

## 样例

### 样例输入 #1
```
2 11
```
### 样例输出 #1
```
5
```
