# [SCOI2005] 超级格雷码
## 题目描述

![](https://cdn.luogu.com.cn/upload/pic/1392.png)

## 输入格式

只有一行，为两个整数n和B。

## 输出格式

一共B^n个行，每行一个B进制数，表示你所求得的符合条件的排列。

## 样例

### 样例输入 #1
```
2  2
```
### 样例输出 #1
```
00
01
11
10

```
