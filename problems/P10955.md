# 正整数拆分
## 题目描述

给定一个正整数 $N$，要求把 $N$ 拆分成若干个正整数相加的形式，参与加法运算的数可以重复。

**注意**:

  * 拆分方案不考虑顺序；
  * 至少拆分成 $2$ 个数的和。

求拆分的方案数 $\bmod 2147483648$ 的结果。
## 输入格式

一个正整数 $N$。
## 输出格式

输出一个整数，表示结果。
## 样例

### 样例输入 #1
```
7
```
### 样例输出 #1
```
14
```
## 提示

$1 \le N \le 4000$
