# 求一次函数解析式
## 题目背景

做数学寒假作业的怨念……

## 题目描述

给定两个整点的坐标，求它们所在直线的函数解析式（一次函数）。

## 输入格式

第一行有两个整数 $x_1,y_1$，表示第一个整点的坐标为 $(x_1,y_1)$。

第二行有两个整数 $x_2,y_2$，表示第二个整点的坐标为 $(x_2,y_2)$。
## 输出格式

输出共一行，即这个函数解析式。

## 样例

### 样例输入 #1
```
3 6
2 4

```
### 样例输出 #1
```
y=2x


```
### 样例输入 #2
```
3 5
2 7
```
### 样例输出 #2
```
y=-2x+11
```
### 样例输入 #3
```
3 -1
5 -2
```
### 样例输出 #3
```
y=-1/2*x+1/2
```
## 提示

$2 x$ 表示为 `2x`。

$\frac{1}{2}x$ 表示为 `1/2*x`。
