# 64位整数乘法
## 题目描述

求 $a$ 乘 $b$ 对 $p$ 取模的值。
## 输入格式

第一行输入整数 $a$，第二行输入整数 $b$，第三行输入整数 $p$。
## 输出格式

输出一个整数，表示 `a*b mod p` 的值。
## 样例

### 样例输入 #1
```
3
4
5
```
### 样例输出 #1
```
2
```
## 提示

$1 \le a,b,p \le 10^{18}$
