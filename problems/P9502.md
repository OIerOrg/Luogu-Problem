# 『MGOI』Simple Round I | A. 魔法数字
## 题目背景

> 目标越坚定的魔法士，越幸运。——殿堂魔法士 S
## 题目描述

初级魔法士小 M 的魔法数字是 $2$。

给定一个正整数 $n$，小 M 需要找到最大的 **偶数** $m$，使得 $2^m<n$。

请你帮帮他。
## 输入格式

一行一个正整数 $n$。
## 输出格式

一行一个 **偶数** $m$，表示答案。
## 样例

### 样例输入 #1
```
7
```
### 样例输出 #1
```
2
```
### 样例输入 #2
```
8
```
### 样例输出 #2
```
2
```
### 样例输入 #3
```
1024
```
### 样例输出 #3
```
8
```
## 提示

**【数据范围】**

对于前 $20\%$ 的数据，$n = 2$。

对于前 $60\%$ 的数据，$n$ 为 $2$ 的幂次。

对于 $100\%$ 的数据，$2 \le n \le 10^9$。
