# [GESP202406 二级] 计数
## 题目描述

小杨认为自己的幸运数是正整数 $k$（注：保证 $1 \le k\le 9$）。小杨想知道，对于从 $1$ 到 $n$ 的所有正整数中， $k$ 出现了多少次。
## 输入格式

第一行包含一个正整数 $n$。

第二行包含一个正整数 $k$。
## 输出格式

输出从 $1$ 到 $n$ 的所有正整数中， $k$ 出现的次数。
## 样例

### 样例输入 #1
```
25
2
```
### 样例输出 #1
```
9
```
## 提示

从 $1$ 到 $25$ 中，$2$ 出现的正整数有 $2,12,20,21,22,23,24,25$ ，一共出现了 $9$ 次。

对于全部数据，保证有 $1 \le n\le 1000,1 \le k\le 9$。
