# 子树和
## 题目描述

给你一棵 $n$ 个结点的树，树的根结点是 $1$。

每个结点的权值都是 $1$，你需要对每个结点 $i$ 求出 $i$ 的子树和，也就是子树中有多少个结点。
## 输入格式

第一行一个数 $n$ 表示树的结点数。

之后 $n-1$ 行，第 $i$ 行输入一个数 $f_{i+1}$ 表示 $i+1$ 点的父亲是 $f_{i+1}$，保证 $f_{i+1}<i+1$。
## 输出格式

输出 $n$ 行，第 $i$ 行一个数表示 $i$ 点的子树和。
## 样例

### 样例输入 #1
```
5
1
2
3
3
```
### 样例输出 #1
```
5
4
3
1
1
```
## 提示

对于 $100\%$ 的数据，满足 $1\le n\le 1000$。
