# [AHOI2004] 数字迷阵
## 题目描述

![](https://cdn.luogu.com.cn/upload/pic/1655.png)

## 输入格式

每行有三个正整数，分别是i,j,m，其中i,j<=10^9,2<=m<=10^4。

## 输出格式

每行输出对应的第i行，第j列的那个正整数对m取模的结果。

## 样例

### 样例输入 #1
```
1 2 99
```
### 样例输出 #1
```
2
```
### 样例输入 #2
```
9 1 999
```
### 样例输出 #2
```
22
```
