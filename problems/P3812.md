# 【模板】线性基
## 题目背景

这是一道模板题。

## 题目描述

给定 $n$ 个整数（数字可能重复），求在这些数中选取任意个，使得他们的异或和最大。

## 输入格式

第一行一个数 $n$，表示元素个数

接下来一行 $n$ 个数

## 输出格式

仅一行，表示答案。

## 样例

### 样例输入 #1
```
2
1 1
```
### 样例输出 #1
```
1
```
### 样例输入 #2
```
4
1 5 9 4

```
### 样例输出 #2
```
13

```
## 提示

$ 1 \leq n \leq 50, 0 \leq S_i < 2 ^ {50} $
