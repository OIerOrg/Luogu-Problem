# 枚举排列（递归实现排列型枚举）
## 题目描述

今有 $n$ 名学生，要从中选出 $k$ 人排成一列拍照。

请按字典序输出所有可能的排列方式。
## 输入格式

仅一行，两个正整数 $n, k$。
## 输出格式

若干行，每行 $k$ 个正整数，表示一种可能的队伍顺序。
## 样例

### 样例输入 #1
```
3 2
```
### 样例输出 #1
```
1 2
1 3
2 1
2 3
3 1
3 2
```
## 提示

对于 $100\%$ 的数据，$1\leq k\leq n \leq 10$。
