# [GESP样题 一级] 闰年求和
## 题目描述

小明刚刚学习了如何判断平年和闰年，他想知道两个年份之间（**不包含起始年份和终止年份**）的闰年年份具体数字之和。你能帮帮他吗？
## 输入格式

输入一行，包含两个整数，分别表示起始年份和终止年份。约定年份在 $1$ 到 $2022$ 之间。
## 输出格式

输出一行，包含一个整数，表示闰年年份具体数字之和。
## 样例

### 样例输入 #1
```
2018 2022
```
### 样例输出 #1
```
2020
```
