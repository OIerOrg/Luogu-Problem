# 反向输出一个三位数
## 题目描述

将一个三位数反向输出，例如输入 $358$，反向输出 $853$。
## 输入格式

一个三位数 $n$。
## 输出格式

反向输出 $n$。
## 样例

### 样例输入 #1
```
100
```
### 样例输出 #1
```
001
```
### 样例输入 #2
```
001
```
### 样例输出 #2
```
100
```
### 样例输入 #3
```
678
```
### 样例输出 #3
```
876
```
