# 数列之异或
## 题目描述

求$1 \bigoplus 2 \bigoplus\cdots\bigoplus N$ 的值。

$A \bigoplus B$ 即$A$ , $B$ 按位异或。

## 输入格式

1 个整数$N$。

## 输出格式

1 个整数，表示所求的值。

## 样例

### 样例输入 #1
```
3
```
### 样例输出 #1
```
0
```
## 提示

• 对于50% 的数据，$1 \le N \le 10^6$；

• 对于100% 的数据，$1 \le N \le 10^{18}$。
