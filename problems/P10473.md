# 表达式计算4
## 题目描述

给出一个表达式，其中运算符仅包含 +，-，*，/，^，要求求出表达式的最终值。

数据可能会出现括号情况，还有可能出现多余括号情况。

数据保证不会出现超过 int 范围的数据，数据可能会出现负数情况。
## 输入格式

仅一行一个字符串，即为表达式。

## 输出格式

仅一行，既为表达式算出的结果。
## 样例

### 样例输入 #1
```
(2+2)^(1+1)
```
### 样例输出 #1
```
16
```
## 提示

表达式总长度不超过 $30$。
