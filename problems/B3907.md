# [语言月赛 202312] NK
## 题目描述

给定两个正整数 $N, K$，请你统计符合以下条件的正整数 $x$ 的数量：

- $1 \leq x \leq N ^ N$。
- $(x \bmod K)$ 是 $N$ 的倍数。
- $x$ 的个位是 $N$。

$x \bmod K$ 代表 $x$ 除以 $K$ 的余数，例如 $7 \bmod 3 = 1$。
## 输入格式

输入一行两个整数 $N, K$。
## 输出格式

输出一行一个整数，代表符合题干中所有条件的 $x$ 的数量。
## 样例

### 样例输入 #1
```
4 6
```
### 样例输出 #1
```
17
```
### 样例输入 #2
```
7 6
```
### 样例输出 #2
```
0
```
## 提示

### 数据规模与约定

- 对于 $30\%$ 的数据，$1 \leq N, K \leq 6$。
- 对于 $100\%$ 的数据，$1 \leq N \leq 9$，$1 \leq K \leq 10 ^ 6$。
