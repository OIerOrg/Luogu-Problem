# 计算并联电阻的阻值
## 题目描述

对于阻值为 $r_1$ 和 $r_2$ 的电阻，其并联电阻阻值公式计算如下：
$$R=\dfrac{1}{\dfrac{1}{r_1}+\dfrac{1}{r_2}}$$

输入实数 $r_1,r_2$。输出并联之后的阻抗大小，结果保留小数点后 $2$ 位。

## 输入格式

两个实数 $r_1,r_2$，以一个空格分开。

## 输出格式

并联之后的阻抗大小，保留小数点后 $2$ 位。

## 样例

### 样例输入 #1
```
1 2
```
### 样例输出 #1
```
0.67
```
