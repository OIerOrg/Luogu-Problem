# 【模板】后缀自动机（SAM）
## 题目描述

给定一个只包含小写字母的字符串 $S$。

请你求出 $S$ 的所有出现次数不为 $1$ 的子串的出现次数乘上该子串长度的最大值。

## 输入格式

一行一个仅包含小写字母的字符串 $S$。
## 输出格式

一个整数，为所求答案。
## 样例

### 样例输入 #1
```
abab
```
### 样例输出 #1
```
4
```
## 提示

对于 $10 \%$ 的数据，$\lvert S \rvert \le 1000$。  
对于 $100\% $的数据，$1 \le \lvert S \rvert \le {10}^6$。
