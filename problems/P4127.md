# [AHOI2009] 同类分布
## 题目描述

给出两个数$a,b$，求出$[a,b]$中各位数字之和能整除原数的数的个数。

## 输入格式

一行，两个整数$a$和$b$

## 输出格式

一个整数，表示答案

## 样例

### 样例输入 #1
```
10 19
```
### 样例输出 #1
```
3
```
## 提示

对于所有的数据，$1 ≤ a ≤ b ≤ 10^{18}$
