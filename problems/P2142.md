# 高精度减法
## 题目描述

高精度减法。
## 输入格式

两个整数 $a,b$（第二个可能比第一个大）。
## 输出格式

结果（是负数要输出负号）。

## 样例

### 样例输入 #1
```
2
1
```
### 样例输出 #1
```
1
```
## 提示

- $20\%$ 数据 $a,b$ 在 long long 范围内；
- $100\%$ 数据 $0<a,b\le 10^{10086}$。
