# BZOJ4316 小 C 的独立集
## 题目描述

给定简单无向图 $G = (V, E)$，保证每条边属于且仅属于一个简单环，求 $G$ 的最大独立集大小。
## 输入格式

第一行输入两个正整数 $n,m$，表示图的点数与边数。

接下来 $m$ 行，每行两个正整数 $u,v$，描述一条无向边。
## 输出格式

输出 $G$ 的最大独立集大小。
## 样例

### 样例输入 #1
```
5 6
1 2
2 3
3 1
3 4
4 5
3 5
```
### 样例输出 #1
```
2
```
## 提示

对于 $100\%$ 的数据，$1\leq n\leq 5\times 10^4$，$1\leq m\leq 6\times 10^4$。
